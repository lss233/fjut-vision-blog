## 如何上传本地文件至doc
1.使用git将项目克隆到本地
```
git clone https://github.com/typical-motion/fjut-vision-blog.git
```

2.在``docs``目录下编写``.md``文件  

**Tips:** 如果需要图片，则需要将图片添加到``static``目录下  
在``.md``文件下使用相对路径  

3.在本地验证是否有bug  
```npm run start```  
4.上传代码至GitHub  
``git push -u origin master``
